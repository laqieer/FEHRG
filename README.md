## Build
1. Put `baserom.gba` in project directory
1. `make`

`FREQ=?`: to control sound sampling frequency/rate (valid: 5734, 7884, 10512, 13379, 15768, 18157, 21024, 26758, 31536, 36314, 40137, 42048) (default: 10512)
